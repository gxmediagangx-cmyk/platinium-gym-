# Project Guidelines & User Preferences

- **Framework:** Next.js (App Router), TypeScript.
- **Styling & UI:** Tailwind CSS, shadcn/ui components.
- **Design:** Dark premium theme, fully responsive design.
- **Architecture:** Component-based architecture with a scalable folder structure.
- **Quality:** High-quality, production-ready, sustainable code that won't easily break.
- **Restrictions:** Do NOT use Angular. Do NOT use plain React without Next.js.
- **Interaction Rule:** If the user asks a question without requesting code changes, only answer the question and do NOT modify the codebase.
